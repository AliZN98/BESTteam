LogMate

LogMate is a flexible software that is able to read and show different log file formats such as: LAS, DLIS, text, CSV, etc. and show well logs data in different views. It allows the user to interpret any log and do depth matching, merging and averaging. Some properties such as porosity and permeability can be calculated in this software and check with core data.

Perforation

This module enables users to design an optimum perforation operation and suggests appropriate perforator guns. It also determines critical production rate and optimum perforation length to prevent water and gas coning.

EcoMate

This module estimates the total cost of the well by considering the costs of different drilling and production operations.Also, this module provides solutions for different problems reported in the desired field or well. And finally, according to the costs of the solutions and flows affected by the problems, it reports the costs of solving the problems.

Water Management

The process of managing the excess water produced in the fields generally begins with the collection of production and reservoir data. For this purpose, this module provides the following abilities: Salinity calculation, Identifying the water production zone by Chan diagram, Providing solution for the water production problem

Mechanical Earth Model

This module builds mechanical earth model for a well using logging and geomechanical data. Different parameters such as elastic parameters, in-situ stresses, UCS, pore and fracture pressure are calculated using various correlations. The output of this module is used for wellbore stability and sand production modeling.

Wellbore Stability

This module detects the failure limits in a directional well using different failure criteria. Geomechanical model is imported from MEM module.The safe drilling interval is also calculated using the failure limit in low and high pressure ranges. There is an option to perform sensitivity analysis on the well deviation and azimuth and get the optimum well deviation to reduce instability problems.

Formation Damage

The outputs of this module are divided into two main categories: advises and calculations Advisor section suggest possible problems based on the user’s answers to the structured questions about well and formation conditions. In each case best remedial actions are suggested to fix or reduce the effect of the problems. In calculation section different sources of skin (damage, turbulency, deviation, partial penetration, etc.) skin in different flow regimes is calculated and the user can perform sensitivity analysis on the effective parameters based on different objectives (such as skin, flow, pressure drop).

Well Integrity

The module provides a framework to help users during the planning and manage the well integrity during operations. User can evaluate risks and production loss by simplifying the well integrity monitoring. This module also presents tools to plot and interpret cement evaluation logs including Acoustic logs (like CBL and VDL), noise logs and etc.

Well Type Analysis

This module enables the user to obtain a meaningful trend of total production at the field, asset, or reservoir level. This module can estimate the field characteristics or the production rate of a new well based on existing well production using P-value distribution rates.

Sand Production

This module allow user to plot critical wellbore pressure versus reservoir pressure chart according to geomechanical data and based on different types of failure models such as Mohr-Coulomb, Mogi-Coulomb, Drucker-Prager and so on to estimate the onset of sand production. Sand production prediction analysis will be done with this module and different graphs such as cumulative sand production, rate of sand production, cumulative sand to fluid production and so on will be plotted. And also, the user can perform sensitivity analysis on the effective parameters.

Well File

The module allows the user to draw the schematic of the well that contains all details (including casings, tubings and various tools such as packers, plugs, downhole pumps, fishes, etc.). It is also possible to change the scale of different parts of the well (for example, magnifying the formation section) to display more details.Also, in this module, boundaries of field and different reservoirs can be specified on the satellite maps and various details can be obtained by using it.

Production

In this module, different type curves are used to analyze and predict the performance of oil and gas wells, and future well production is estimated based on. This estimation includes all types of production phases in the reservoir, including single-phase and two-phase.The methods used in this module can be divided into Arps experimental method, Fetkovich type curves, Analytical Transient method, and Locke and Sawyer method.

Pressure Analysis

This module provides a continuous process of pressure data in terms of production data (gas or oil or water production), using a combination of reservoir pressure data, the pressure calculation function and the P/Z line. Based on this process, it is possible to estimate the initial gas volume in the reservoir (GIIP).
